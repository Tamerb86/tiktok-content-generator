## 📊 API Endpoints

### User Routes
- ✅ `GET /api/v1/me` - User profile + usage stats
- ✅ `PATCH /api/v1/me` - Update profile

### Product Routes
- ✅ `GET /api/v1/products` - List products (paginated)
- ✅ `POST /api/v1/products` - Create product
- ✅ `GET /api/v1/products/:id` - Get product + last content
- ✅ `PATCH /api/v1/products/:id` - Update product
- ✅ `DELETE /api/v1/products/:id` - Delete product

### Generate Routes
- ✅ `POST /api/v1/generate/full-package` - Generate content
- ✅ `GET /api/v1/generate/usage` - Usage stats
- ✅ `GET /api/v1/generate/history` - Generation history
- ✅ `GET /api/v1/generate/content/:id` - Get content
- ✅ `GET /api/v1/generate/product/:productId` - Product contents
- ✅ `DELETE /api/v1/generate/content/:id` - Delete content
- ✅ `GET /api/v1/generate/options` - Available options

### Plans Routes
- ✅ `GET /api/v1/plans` - List plans
- ✅ `GET /api/v1/plans/:code` - Get plan details

### Billing Routes
- ✅ `POST /api/v1/billing/webhook` - Stripe webhook
- ✅ `POST /api/v1/billing/create-checkout-session` - Create checkout
- ✅ `POST /api/v1/billing/create-portal-session` - Customer portal
- ✅ `GET /api/v1/billing/subscription` - Subscription status
- ✅ `POST /api/v1/billing/cancel` - Cancel subscription
- ✅ `POST /api/v1/billing/resume` - Resume subscription
- ✅ `GET /api/v1/billing/usage` - Usage statistics

### API Keys Routes
- ✅ `GET /api/v1/api-keys` - List API keys
- ✅ `POST /api/v1/api-keys` - Create API key
- ✅ `PATCH /api/v1/api-keys/:id` - Update label
- ✅ `POST /api/v1/api-keys/:id/deactivate` - Deactivate
- ✅ `POST /api/v1/api-keys/:id/reactivate` - Reactivate
- ✅ `DELETE /api/v1/api-keys/:id` - Delete
