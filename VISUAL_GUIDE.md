## 🔄 4. سير العمل الكامل (Complete Flow)

```
┌──────────────────────────────────────────────────────────────┐
│  1. المستخدم يفتح صفحة منتج على AliExpress/Amazon           │
└────────────────────────┬─────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────────┐
│  2. إضافة Chrome تستخرج البيانات تلقائياً:                  │
│     • العنوان                                                │
│     • الوصف                                                  │
│     • الصور                                                  │
│     • السعر                                                  │
│     • الميزات                                                │
└────────────────────────┬─────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────────┐
│  3. المستخدم يختار الإعدادات في Popup:                      │
│     • اللغة (عربي/إنجليزي)                                  │
│     • المنصة (TikTok/Reels/Shorts)                           │
│     • النبرة (ودود/احترافي/عاجل)                            │
│     • المجال (إلكترونيات/أزياء/تجميل)                       │
└────────────────────────┬─────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────────┐
│  4. الإضافة ترسل POST request للـ Backend:                  │
│     POST /api/v1/products (حفظ المنتج)                      │
│     POST /api/v1/generate/full-package (توليد المحتوى)      │
│     Header: X-API-Key: tcg_xxxxxxxxxxxxx                     │
└────────────────────────┬─────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────────┐
│  5. Backend يتحقق من:                                        │
│     ✓ صحة API Key                                            │
│     ✓ حدود الاستخدام (5/10 used)                            │
│     ✓ حالة الاشتراك (active)                                │
└────────────────────────┬─────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────────┐
│  6. OpenAI يولد المحتوى الكامل:                             │
│     • Script (20-30s)                                        │
│     • 10 Hooks                                               │
│     • 5 Creative Angles                                      │
│     • 5 Captions                                             │
│     • 20 Hashtags                                            │
│     • 3 Thumbnail Texts                                      │
└────────────────────────┬─────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────────┐
│  7. Backend يحفظ:                                            │
│     • المنتج في جدول products                                │
│     • المحتوى في جدول generated_contents                     │
│     • الاستهلاك في جدول usage_logs                           │
└────────────────────────┬─────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────────┐
│  8. الإضافة تعرض:                                            │
│     • المحتوى المولد في Popup                                │
│     • أو رابط Dashboard لعرض تفصيلي                         │
│     • زر Copy لكل قسم                                        │
└──────────────────────────────────────────────────────────────┘
```
